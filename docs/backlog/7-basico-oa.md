## Prioridades para 7° Básico

### Fase 1: Usar Lecciones M1 Existentes
La mayoría de OA de 7° básico ya están cubiertos por lecciones M1. Un estudiante de 7° puede usar:
- Todas las lecciones de números (m1-num-001 a m1-num-005)
- Lecciones de álgebra básica (m1-alg-002 a m1-alg-006)
- Lecciones de círculo (m1-geo-002)
- Lecciones de probabilidad básica (m1-prob-001, m1-prob-002, m1-prob-004)

### Fase 2: Desarrollar Lecciones Específicas 7°B
Solo si el proyecto decide expandirse a 7° básico específicamente:
