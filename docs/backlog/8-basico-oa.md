## Cross-Referencias con M1

8° Básico es el puente directo hacia PAES M1. La mayoría de contenidos de 8° son prerrequisitos directos:

```
┌────────────────────────────────────────────────────────────────┐
│                    8° BÁSICO (Consolidación)                   │

│  MA08-OA-06    │  MA08-OA-07    │  MA08-OA-12    │  MA08-OA-17 │
│  Factorización │  Func. Lineal  │  Pitágoras     │  Conteo     │
└────────────────────────────────────────────────────────────────┘
         │               │               │               │
         ▼               ▼               ▼               ▼
┌────────────────────────────────────────────────────────────────┐
│                    PAES M1 (Aplicación)                        │
├────────────────────────────────────────────────────────────────┤
│  m1-alg-001    │  m1-alg-009    │  m1-geo-001    │  m1-prob-005│
│  10 lecciones  │  Func. Afín    │  Pitágoras     │  Conteo     │
└────────────────────────────────────────────────────────────────┘
```

### Hallazgo Clave

8° Básico tiene **76% de cobertura** con las lecciones M1 existentes porque:
1. Es el año de consolidación antes de enseñanza media
2. Los OA de 8° son casi idénticos a los fundamentos de PAES M1
3. Las brechas de 8° son las mismas brechas identificadas en M1 (transformaciones, volúmenes, inecuaciones)

## Prioridades para 8° Básico

### Fase 1: Las lecciones M1 ya sirven para 8° Básico
Un estudiante de 8° básico puede usar directamente:
- Toda la unidad M1-ALG-001 (factorización)
- Lecciones de función lineal (m1-alg-009)
- Teorema de Pitágoras (m1-geo-001-a)
- Lecciones de probabilidad y conteo (m1-prob-004, m1-prob-005)

### Fase 2: Completar brechas M1 (beneficia 8° y PAES)
Estas lecciones pendientes cubren tanto 8° básico como PAES:

| Lecciones M1 Pendientes | Cubre OA 8° | Cubre PAES |
|------------------------|-------------|------------|
| m1-alg-006-b/m1-alg-007-a | MA08-OA-09 | Sí |
| m1-geo-003-b/c/d | MA08-OA-11 | Sí |
| m1-geo-005-a/b/c/d | MA08-OA-13/14 | Sí |

### Fase 3: Lecciones específicas 8° (opcional)
Solo si el proyecto decide crear contenido específico para 8°:
- 8b-prob-001: Análisis Crítico de Gráficos
- m2-num-003-a: Interés Simple
