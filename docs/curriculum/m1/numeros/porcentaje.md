# Porcentajes

## Overview

Los porcentajes son una forma de expresar números como fracciones de 100. Son fundamentales para resolver problemas de la vida cotidiana y aparecen frecuentemente en la PAES.


---

## Conceptos Clave

### ¿Qué es un Porcentaje?

Un porcentaje representa una fracción cuyo denominador es 100. El símbolo % significa "por ciento" o "de cada 100".

$$x\% = \frac{x}{100}$$

**Ejemplos**:
$$25\% = \frac{25}{100} = 0.25$$
$$50\% = \frac{50}{100} = \frac{1}{2} = 0.5$$
$$100\% = \frac{100}{100} = 1$$

### Conversiones Básicas

#### Porcentaje a Decimal
Dividir por 100 (mover la coma dos lugares a la izquierda):
$$35\% = 0.35$$
$$7.5\% = 0.075$$

#### Decimal a Porcentaje
Multiplicar por 100 (mover la coma dos lugares a la derecha):
$$0.45 = 45\%$$
$$1.2 = 120\%$$

#### Porcentaje a Fracción
$$40\% = \frac{40}{100} = \frac{2}{5}$$
$$75\% = \frac{75}{100} = \frac{3}{4}$$

---

## Operaciones con Porcentajes

### Calcular el Porcentaje de una Cantidad

**Fórmula**:
$$\text{Resultado} = \frac{\text{Porcentaje}}{100} \times \text{Cantidad Total}$$

**Ejemplo**: ¿Cuánto es el 30% de 200?
$$30\% \text{ de } 200 = \frac{30}{100} \times 200 = 0.3 \times 200 = 60$$

### Encontrar qué Porcentaje es una Cantidad de Otra

**Fórmula**:
$$\text{Porcentaje} = \frac{\text{Parte}}{\text{Total}} \times 100\%$$

**Ejemplo**: ¿Qué porcentaje de 80 es 20?
$$\frac{20}{80} \times 100\% = 0.25 \times 100\% = 25\%$$

### Aumentos Porcentuales

**Fórmula**:
$$\text{Valor Final} = \text{Valor Inicial} \times \left(1 + \frac{\text{Aumento}\%}{100}\right)$$

**Ejemplo**: Un producto cuesta $1000. Si aumenta un 15%, ¿cuál es el nuevo precio?
$$1000 \times \left(1 + \frac{15}{100}\right) = 1000 \times 1.15 = 1150$$

### Disminuciones Porcentuales

**Fórmula**:
$$\text{Valor Final} = \text{Valor Inicial} \times \left(1 - \frac{\text{Descuento}\%}{100}\right)$$

**Ejemplo**: Un producto de $500 tiene un descuento del 20%. ¿Cuál es el precio final?
$$500 \times \left(1 - \frac{20}{100}\right) = 500 \times 0.8 = 400$$

---

## Aumentos y Descuentos Sucesivos

**IMPORTANTE**: Dos aumentos del 10% NO equivalen a un aumento del 20%.

**Ejemplo**: Un precio de $100 aumenta 10% y luego aumenta otro 10%:
$$\text{Primer aumento: } 100 \times 1.10 = 110$$
$$\text{Segundo aumento: } 110 \times 1.10 = 121$$

El aumento total es del 21%, no del 20%.

**Fórmula general**:
$$\text{Valor Final} = \text{Valor Inicial} \times (1 + r_1) \times (1 + r_2)$$

donde $r_1$ y $r_2$ son las tasas en forma decimal.

## Porcentajes Importantes

### Porcentajes Comunes

| Porcentaje | Fracción | Decimal |
|-----------|----------|---------|
| 1% | $\frac{1}{100}$ | 0.01 |
| 10% | $\frac{1}{10}$ | 0.1 |
| 20% | $\frac{1}{5}$ | 0.2 |
| 25% | $\frac{1}{4}$ | 0.25 |
| 50% | $\frac{1}{2}$ | 0.5 |
| 75% | $\frac{3}{4}$ | 0.75 |
| 100% | $1$ | 1.0 |

### Porcentajes Mayores a 100%

Un porcentaje mayor a 100% significa que la cantidad es mayor que el total de referencia.

$$150\% = 1.5 = \frac{3}{2}$$

**Ejemplo**: Si una población aumentó 150%, significa que ahora es 2.5 veces la población original.

## Ejemplos Tipo PAES

### Ejemplo 1: Cálculo Básico

**Pregunta**: En una clase de 40 estudiantes, el 35% son mujeres. ¿Cuántas mujeres hay?

**Opciones**:
- A) 12
- B) 14
- C) 16
- D) 18

**Solución**:
$$35\% \text{ de } 40 = \frac{35}{100} \times 40 = 0.35 \times 40 = 14$$

**Respuesta**: B) 14

---

### Ejemplo 2: Descuento

**Pregunta**: Un televisor cuesta $450.000. Si tiene un descuento del 20%, ¿cuál es el precio con descuento?

**Opciones**:
- A) $350.000
- B) $360.000
- C) $380.000
- D) $430.000

**Solución**:
$$\text{Descuento} = 450.000 \times 0.20 = 90.000$$
$$\text{Precio final} = 450.000 - 90.000 = 360.000$$

**Método alternativo**:
$$450.000 \times (1 - 0.20) = 450.000 \times 0.8 = 360.000$$

**Respuesta**: B) $360.000

---

### Ejemplo 3: Encontrar el Porcentaje

**Pregunta**: En un examen de 60 preguntas, María respondió correctamente 45. ¿Qué porcentaje de respuestas correctas obtuvo?

**Opciones**:
- A) 65%
- B) 70%
- C) 75%
- D) 80%

**Solución**:
$$\frac{45}{60} \times 100\% = 0.75 \times 100\% = 75\%$$

**Respuesta**: C) 75%

---

### Ejemplo 4: Aumentos Sucesivos

**Pregunta**: Un producto de $200 aumenta 10% y luego aumenta otro 10%. ¿Cuál es el precio final?

**Opciones**:
- A) $220
- B) $240
- C) $242
- D) $250

**Solución**:
$$\text{Primer aumento: } 200 \times 1.10 = 220$$
$$\text{Segundo aumento: } 220 \times 1.10 = 242$$

**Respuesta**: C) $242

---

### Ejemplo 5: Porcentaje de Cambio

**Pregunta**: El precio de un producto bajó de $80 a $64. ¿Cuál fue el porcentaje de disminución?

**Opciones**:
- A) 16%
- B) 20%
- C) 25%
- D) 30%

**Solución**:
$$\text{Cambio} = 80 - 64 = 16$$
$$\text{Porcentaje} = \frac{16}{80} \times 100\% = 0.2 \times 100\% = 20\%$$

**Respuesta**: B) 20%

## Errores Comunes

### ❌ Confundir Base de Cálculo

**Error**: Calcular el 30% de 50 como $30 \times 50 = 1500$
$$\text{✓ Correcto: } \frac{30}{100} \times 50 = 15$$

### ❌ Sumar Porcentajes Directamente

**Error**: Dos aumentos del 15% = aumento del 30%
$$\text{❌} \quad 100 \rightarrow \text{+30%} = 130$$
$$\text{✓} \quad 100 \times 1.15 \times 1.15 = 132.25$$

### ❌ Olvidar que el Porcentaje es de 100

**Error**: Escribir 25% como 25 en cálculos
$$\text{❌} \quad 25 \times 80 = 2000$$
$$\text{✓} \quad 0.25 \times 80 = 20$$

### ❌ Usar la Base Incorrecta para Cambio Porcentual

**Error**: Si el precio sube de $50 a $60, calcular $\frac{10}{60} \times 100\%$
$$\text{✓ Correcto: } \frac{10}{50} \times 100\% = 20\%$$

La base debe ser el valor **inicial**, no el final.

## Aplicaciones Prácticas

### Impuestos
IVA del 19%:
$$\text{Precio con IVA} = \text{Precio} \times 1.19$$

### Propinas
Propina del 10%:
$$\text{Propina} = \text{Cuenta} \times 0.10$$

### Intereses
Interés simple del 5% anual:
$$\text{Interés} = \text{Capital} \times 0.05$$

### Estadísticas
Tasa de aprobación:
$$\text{Tasa} = \frac{\text{Aprobados}}{\text{Total}} \times 100\%$$
