# Expresiones Algebraicas

## Overview

Las expresiones algebraicas son combinaciones de números, letras (variables) y operaciones matemáticas. Son el lenguaje fundamental del álgebra y permiten generalizar patrones y resolver problemas.


---

## Conceptos Clave

### ¿Qué es una Expresión Algebraica?

Una expresión algebraica es una combinación de constantes, variables y operaciones.

**Ejemplos**:
$$3x + 5$$
$$2a^2 - 4ab + b^2$$
$$\frac{x+1}{x-2}$$

### Componentes

**Término**: Expresión separada por suma o resta
$$3x^2 - 5x + 7 \quad \text{(3 términos)}$$

**Coeficiente**: Número que multiplica a la variable
$$\underbrace{5}_{coef}x^2$$

**Variable**: Letra que representa un valor desconocido
$$3\underbrace{x}_{var} + 2\underbrace{y}_{var}$$

**Constante**: Término sin variable
$$2x + \underbrace{7}_{const}$$

**Exponente**: Potencia de la variable
$$x\underbrace{^3}_{exp}$$

---

## Tipos de Expresiones

### Monomio
Un solo término:
$$5x^2, \quad -3ab, \quad 7$$

### Binomio
Dos términos:
$$x + 5, \quad 3a - 2b, \quad x^2 + 1$$

### Trinomio
Tres términos:
$$x^2 + 5x + 6, \quad a^2 - 2ab + b^2$$

### Polinomio
Varios términos:
$$4x^3 - 2x^2 + 7x - 3$$

---

## Grado de una Expresión

### Grado de un Monomio
Suma de los exponentes de todas las variables:

$$5x^3 \quad \rightarrow \quad \text{Grado 3}$$
$$2x^2y \quad \rightarrow \quad \text{Grado } 2+1 = 3$$
$$-7xy^2z \quad \rightarrow \quad \text{Grado } 1+2+1 = 4$$

### Grado de un Polinomio
El mayor grado entre sus términos:

$$3x^4 + 2x^2 - 5x + 1 \quad \rightarrow \quad \text{Grado 4}$$

---

## Valor Numérico

Evaluar una expresión es sustituir las variables por números específicos.

**Ejemplo**: Evaluar $3x^2 - 2x + 5$ cuando $x = 2$

$$\begin{align}
&= 3(2)^2 - 2(2) + 5\\
&= 3(4) - 4 + 5\\
&= 12 - 4 + 5\\
&= 13
\end{align}$$

---

## Términos Semejantes

Términos que tienen las mismas variables con los mismos exponentes.

**Semejantes**:
$$3x^2 \text{ y } -5x^2 \quad \checkmark$$
$$2xy \text{ y } 7xy \quad \checkmark$$

**NO semejantes**:
$$3x^2 \text{ y } 3x \quad \text{(exponentes diferentes)}$$
$$2xy \text{ y } 2x \quad \text{(variables diferentes)}$$

---

## Operaciones con Expresiones Algebraicas

### Suma y Resta

**Regla**: Solo se pueden sumar/restar términos semejantes.

**Ejemplo**:
$$(3x^2 + 5x - 2) + (2x^2 - 3x + 7)$$

$$\begin{align}
&= 3x^2 + 2x^2 + 5x - 3x - 2 + 7\\
&= 5x^2 + 2x + 5
\end{align}$$

### Multiplicación de Monomios

Multiplicar coeficientes y sumar exponentes de la misma base.

**Ejemplo**:
$$(3x^2)(4x^3) = 12x^{2+3} = 12x^5$$

$$(2xy)(-5x^2y) = -10x^{1+2}y^{1+1} = -10x^3y^2$$

### Propiedad Distributiva

$$a(b + c) = ab + ac$$

**Ejemplo**:
$$3x(2x + 5) = 3x \cdot 2x + 3x \cdot 5 = 6x^2 + 15x$$

---

## Productos Notables

### Cuadrado de un Binomio

$$(a + b)^2 = a^2 + 2ab + b^2$$
$$(a - b)^2 = a^2 - 2ab + b^2$$

**Ejemplos**:
$$(x + 3)^2 = x^2 + 2(x)(3) + 3^2 = x^2 + 6x + 9$$

$$(2x - 5)^2 = (2x)^2 - 2(2x)(5) + 5^2 = 4x^2 - 20x + 25$$

### Suma por Diferencia

$$(a + b)(a - b) = a^2 - b^2$$

**Ejemplo**:
$$(x + 4)(x - 4) = x^2 - 16$$

$$(3x + 2)(3x - 2) = 9x^2 - 4$$

### Producto de Binomios con Término Común

$$(x + a)(x + b) = x^2 + (a+b)x + ab$$

**Ejemplo**:
$$(x + 3)(x + 5) = x^2 + 8x + 15$$

$$(x - 2)(x + 7) = x^2 + 5x - 14$$

---

## Factorización

Proceso inverso de multiplicar: expresar como producto.

### Factor Común

$$ax + ay = a(x + y)$$

**Ejemplo**:
$$6x^2 + 9x = 3x(2x + 3)$$

### Diferencia de Cuadrados

$$a^2 - b^2 = (a+b)(a-b)$$

**Ejemplo**:
$$x^2 - 25 = (x+5)(x-5)$$

$$9x^2 - 16 = (3x+4)(3x-4)$$

### Trinomio Cuadrado Perfecto

$$a^2 + 2ab + b^2 = (a+b)^2$$
$$a^2 - 2ab + b^2 = (a-b)^2$$

**Ejemplo**:
$$x^2 + 6x + 9 = (x+3)^2$$

$$4x^2 - 12x + 9 = (2x-3)^2$$

### Trinomio de la forma $x^2 + bx + c$

Buscar dos números que sumen $b$ y multipliquen $c$.

**Ejemplo**:
$$x^2 + 7x + 12$$

Buscar números que sumen 7 y multipliquen 12: **3 y 4**

$$x^2 + 7x + 12 = (x+3)(x+4)$$

## Ejemplos Tipo PAES

### Ejemplo 1: Reducir Términos Semejantes

**Pregunta**: Simplifica $5x + 3y - 2x + 7y$

**Opciones**:
- A) $3x + 10y$
- B) $7x + 4y$
- C) $3x + 4y$
- D) $7x + 10y$

**Solución**:
$$5x - 2x + 3y + 7y = 3x + 10y$$

**Respuesta**: A) $3x + 10y$

---

### Ejemplo 2: Producto Notable

**Pregunta**: ¿Cuál es el resultado de $(x+5)^2$?

**Opciones**:
- A) $x^2 + 25$
- B) $x^2 + 5x + 25$
- C) $x^2 + 10x + 25$
- D) $x^2 + 10x + 5$

**Solución**:
$$(x+5)^2 = x^2 + 2(x)(5) + 5^2 = x^2 + 10x + 25$$

**Respuesta**: C) $x^2 + 10x + 25$

---

### Ejemplo 3: Factorización

**Pregunta**: Factoriza $x^2 - 9$

**Opciones**:
- A) $(x-3)^2$
- B) $(x+3)^2$
- C) $(x+3)(x-3)$
- D) $(x-9)(x+1)$

**Solución**:
$$x^2 - 9 = x^2 - 3^2 = (x+3)(x-3)$$

**Respuesta**: C) $(x+3)(x-3)$

---

### Ejemplo 4: Valor Numérico

**Pregunta**: Si $x=3$, ¿cuál es el valor de $2x^2 - 5x + 1$?

**Opciones**:
- A) 4
- B) 8
- C) 10
- D) 16

**Solución**:
$$2(3)^2 - 5(3) + 1 = 2(9) - 15 + 1 = 18 - 15 + 1 = 4$$

**Respuesta**: A) 4

---

### Ejemplo 5: Multiplicación

**Pregunta**: ¿Cuál es el resultado de $(x+2)(x+3)$?

**Opciones**:
- A) $x^2 + 5x + 6$
- B) $x^2 + 6x + 5$
- C) $x^2 + 5x + 5$
- D) $x^2 + 6x + 6$

**Solución**:
$$(x+2)(x+3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6$$

**Respuesta**: A) $x^2 + 5x + 6$

---

## Errores Comunes

### ❌ $(a+b)^2 \neq a^2 + b^2$

$$\text{❌} \quad (x+3)^2 = x^2 + 9$$
$$\text{✓} \quad (x+3)^2 = x^2 + 6x + 9$$

### ❌ Sumar términos no semejantes

$$\text{❌} \quad 3x + 2x^2 = 5x^3$$
$$\text{✓} \quad 3x + 2x^2 \text{ (no se puede simplificar)}$$

### ❌ Olvidar el signo negativo

$$\text{❌} \quad -(x-3) = -x - 3$$
$$\text{✓} \quad -(x-3) = -x + 3$$

### ❌ Confundir coeficiente con exponente

$$\text{❌} \quad (2x)^2 = 2x^2$$
$$\text{✓} \quad (2x)^2 = 4x^2$$

## Lenguaje Algebraico

Traducir frases del lenguaje común al algebraico:

| Frase | Expresión |
|-------|-----------|
| El doble de un número | $2x$ |
| Un número aumentado en 5 | $x + 5$ |
| El cuadrado de un número | $x^2$ |
| La mitad de un número | $\frac{x}{2}$ |
| Tres números consecutivos | $x, x+1, x+2$ |
| El producto de dos números | $xy$ |
