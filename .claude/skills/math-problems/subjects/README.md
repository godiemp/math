# Subject Selection Guide

Before creating questions, identify the mathematical subject and read the corresponding template file.

## Decision Tree

```
What is the question about?
         |
    +----+----+----+----+
    |    |    |    |
    v    v    v    v

EXPRESSIONS,    FRACTIONS,     SHAPES,      DATA,
EQUATIONS,      PERCENTAGES,   AREAS,       GRAPHS,
FUNCTIONS,      DECIMALS,      VOLUMES,     COUNTING
FACTORING       RATIOS         THEOREMS     PROBABILITY
    |              |             |              |
    v              v             v              v
  ALGEBRA       NUMEROS     GEOMETRIA    PROBABILIDAD
```

## Subject Files

| Subject | File | Key Topics |
|---------|------|------------|
| Algebra | `algebra-templates.md` | Expressions, equations, functions, factoring |
| Numeros | `numeros-templates.md` | Fractions, percentages, operations, ratios |
| Geometria | `geometria-templates.md` | Areas, perimeters, volumes, theorems |
| Probabilidad | `probabilidad-templates.md` | Probability, statistics, data, graphs |

## Topic String by Subject

| Subject | topic String |

## Common Contexts by Subject

### Algebra
- Business: inventory, profits, costs
- Science: concentrations, physics formulas
- Construction: measurements, materials
- Technology: file sizes, storage

### Numeros
- Shopping: prices, discounts, taxes
- Cooking: recipes, portions
- Finance: savings, interest
- Travel: distances, time, speed

### Geometria
- Construction: buildings, cables, ramps
- Design: logos, patterns
- Navigation: distances, coordinates
- Sports: field dimensions

### Probabilidad
- Surveys: preferences, opinions
- Games: dice, cards, spinners
- Quality control: defects, samples
- Weather: predictions, frequencies
